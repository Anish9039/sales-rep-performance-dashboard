# Lightweight Sales Rep Dashboard

A mobile-friendly, rep-facing dashboard built with Python and Streamlit.

### Features
*   **Privacy Focus:** Filtered views ensuring reps only see their own data.
*   **Performance:** Caching implemented for fast loading on mobile.
*   **KPI Tracking:** Real-time calculation of "Distance to Goal" and Monthly Progress.

### Tech Stack
*   Python 3.9+
*   Streamlit
*   Pandas
